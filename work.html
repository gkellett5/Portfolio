Weather App

Breif: Create a fully operational weather app (no fake data). The project should include a search engine, API integration, wind speed, weather description, and weather icons. The forecast is optional. Host your project on Netlify and add a link to the Github repository.
This app was my final project for the SheCodes Plus workshop, it challenged my HTML and CSS skills by adding API integration and a search engine. I went above what was asked and made my weather app open with a geolocation, allowing you to see your local weather and forecast. For this project we were also requred to build a wireframe and were given complete freedom on the design, should we choose. [open in Chrome]
